Sistema de Gestión de Carrito de Compras y Pedido

Python

programa que permite a los usuarios registrarse, iniciar sesión y navegar por un catálogo de productos organizado por categorías. Ofrece funciones para agregar productos al carrito, modificar cantidades, eliminar artículos y realizar pedidos con cálculo automático de totales. Además, gestiona inventarios, genera fechas de entrega aleatorias y almacena toda la información en una base de datos relacional para mantener un flujo completo de compra en línea.
Tecnologías: Python y bases de datos relacionales (SQL).

Pagina Web Haloween

Html, Css

La página web ofrece una experiencia de Halloween con atracciones terroríficas donde los usuarios pueden conocer a los monstruos que habitan cada atracción, ver fotos y leer información sobre ellos. Además, permite comprar tickets para las atracciones, brindando detalles como la temática, duración e intensidad del terror. Todo esto en una interfaz emocionante y fácil de usar.
Tecnologías: Python y bases de datos relacionales (SQL).
